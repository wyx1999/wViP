Tutorial

1. Data loading

       wViP supports users to input text directly through the keyboard and can upload files in three formats: Microsoft Word (07-2019, *.docx), Portable Document Format (*.pdf) and Text File (*.txt).


2. Advance setting

2.1 Dictionary

       Set specific phrases for natural language processing. The website provides default dictionaries for many majors and fields, including biology, medicine, etc. Users can import, modify and upload customized dictionaries (*.txt) in the dictionary option.

2.2 Stop words

       Set the stop word list to filter out unwanted messages. The default stop list contains determiners, prepositions, pronouns, etc. You can modify, export, and clear the default list. The words in the stop word list will not appear in the natural language processing results and word cloud.

2.3 Case of word

       To set the case of words in natural language processing, three options are provided: sensitive (keep the text input style unchanged) and insensitive (all letters are lowercase).

In the case sensitive option, the case of the same word is treated as different words in the word frequency statistics.


3. Natural language processing

3.1 Tokenizer

       Word segmentation of the input text data according to the set dictionary and stop word list.

3.2 Part-Of-Speech tagging

       Determine the part of speech of a word according to the corpus and the position of the word in the sentence. Divide words into four categories: noun, verb, adjective and adverb. For words with more than one part of speech, the user can adjust as needed in the NLP results adjustment option.

3.3 Lemmatization

       After word segmentation, each word is restored to the prototype in the dictionary according to the part-of-speech tagging.

3.4 Word frequency

       After lemmatization, count the number of occurrences of each word. In the NLP results adjustment option, users can adjust the word frequency results and delete the unwanted words.


4. Word cloud

4.1 Static Visualization

       Draw a word cloud according to the natural language processing results to show the theme and keywords of the text. Word font size is positively correlated with word frequency. To enhance the visual effect, users can select the font color to change with the word frequency. In addition, the Word cloud option provides a variety of fonts and masks to beautify and adapt to various application scenarios.

4.2 Interactive Visualization

       In order to intuitively display the importance of different words in the word cloud, wViP provides interactive visualization. The user can move the mouse over the image to view the word information.


5. Enrichment analysis

5.1 Data loading

       On the enrichment analysis page, users can input UniProt IDs through keyboard input and file upload(*.txt).

5.2 Enrichment analysis

       wViP can be used for enrichment analysis of 12 model species, including Homo sapiens, Mus musculus, Rattus norvegicus, Saccharomyces cerevisiae, Drosophila melanogaster, Arabidopsis thaliana, Sus scrofa, Canis lupus familiaris, Bos taurus, Gallus gallus, Caenorhabditis elegans, and Danio rerio.

       wViP compiled 15 types of functional annotations, including 4 sets of Gene Ontology (GO) annotations (All, biology processes, molecular functions, and cellular components), Kyoto Encyclopedia of Genes and Genomes (KEGG) pathways, Disease Ontology (DO) terms, and 9 sets of Hallmark Gene Sets taken from Molecular signatures database (MSigDB) (Positional Gene Sets, Curated Gene Sets, Regulatory Target Gene Sets, Computational Gene Sets, Ontology Gene Sets, Oncogenic Signature Gene Sets, Immunologic Signature Gene Sets, Cell Type Signature Gene Sets, and All Gene Sets mixed). For the enrichment analysis, the hypergeometric test was used to calculate an enrichment ratio (E-ratio) and a P value for each category of functional annotations.

5.3 Visualization of results

       In the visualization options, two enrichment analysis results display methods (Treemap and Funnel plot) are provided. Users can correlate the font size and color extent with E-ratio or P value respectively.

FAQ

1. Q: What does n, m, N, M, E-ratio, P value mean in the enrichment results?

A: N = number of proteins in background annotated by at least one GO term
    n = number of proteins in background annotated by the GO term t
    M = number of proteins in foreground annotated by at least one GO term
    m = number of proteins in foreground annotated by the GO term t
    E-ratio=(m/M)/(n/N)
    P value was calculated with the hypergeometric distribution

2. Q: When I select both the Color varieties with Mask and Color varieties with font size options, which color range will be displayed?

A: The Color varies with font size option has a higher priority than the Color varies with Mask option. Once it was ticked, the Color varies with Mask option would be invalid. So, the color range would vary with font size.
